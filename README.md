> ### Usage:
> ```
> $ tstr <text>
> ```
>
> ### Example:
> ```
> $ tstr The quick brown fox, jumps over the lazy dog.
> ```
>
> \* There is no help command and no flags. Usage is displayed when running `$ tstr` alone.
